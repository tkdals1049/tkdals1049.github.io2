GDI를 이용한 록맨제로 개발
2017년 7월-2017년 8월
Tools Used: GDI, C++
#록맨 제로 #GDI게임
많은 사람들이 즐겨왔던 록맨제로를 프로그래밍으로 재현하는 것을 목표로 한다. 스프라이트와 GDI를 이용하여 록맨제로의 디자인을 구현하고 이를 기반으로 2D 로그라이크 게임을 만든다. 지면,캐릭터,피격 등의 충돌을 스무스하게 작용하도록 만든다.

Tools Used: DieretX, C++, API

3D 액션 어드벤처 게임 정석이라고도 할 수 있는 몬스터 헌터를 DirectX와 C++로 구현하는 것을 목표로 한다. 몬스터헌터에서 직접 모델과 사운드 등의 데이터를 뽑아와 재구성함으로 몬스터헌터를 재현하도록 한다. 이외에도 맵이나 캐릭터를 설정하고 저장이 가능한 에디터를 만들어 개발툴을 만들어 더 많은 기능을 추가가 가능하도록 한다.
